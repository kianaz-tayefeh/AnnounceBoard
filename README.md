# Announce-Board

Welcome to Announce-Board, a Vite React project built with Mantine and TypeScript.

## Getting Started

`yarn install`

`yarn dev` and `yarn test`

## Packages Used

- `mantine/core`
- `mantine/form`
- `jotai`
- `date-fns`
- `uuid`

## Code Style

Configured prettier, eslint and huskey
